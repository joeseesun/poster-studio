# 乔木画布 - 使用指南

## 快速开始

### 1. 启动应用

```bash
npm install
npm run dev
```

访问 http://localhost:3000

### 2. 界面介绍

```

│ 乔木画布                         [下载] [复制]         │
├────────────────────────────────────────────────────────┤
│ [版本1] [版本2] [版本3] [+ 复制版本]                  │
├──────────┬─────────────────────────┬───────────────────┤
│          │                         │                   │
│ 工具栏   │      Canvas 画布        │   属性面板        │
```

## 基础操作

### 添加普通文本

1. 点击左侧工具栏的 **"+ 添加文本"** 按钮
2. 画布中央会出现一个文本框，显示 "双击编辑"
3. 双击文本框进入编辑模式
4. 输入你想要的文字
5. 点击画布空白处完成编辑

### 添加高亮文字

1. 在右侧属性面板的文本框中输入文字
2. 选择高亮样式：
   - **荧光笔**: 半透明彩色背景
   - **下划线**: 文字下方彩色线条
   - **边框**: 文字周围彩色边框
3. 选择颜色（5 种预设颜色）
4. 点击 **"添加到画布"** 按钮

### 编辑文字样式

1. 点击选中画布上的文字
2. 在右侧属性面板中调整：
   - **字号**: 拖动滑块调整（20-120）
   - **字体**: 下拉选择（思源黑体/思源宋体/Arial）
   - **颜色**: 点击颜色选择器

### 移动和缩放

- **移动**: 拖动对象
- **缩放**: 拖动对象四角的控制点
- **旋转**: 拖动对象顶部的旋转控制点
- **删除**: 选中对象后，点击右侧的 "删除" 按钮

## 版本管理

### 创建新版本

1. 点击版本栏右侧的 **"+ 复制版本"** 按钮
2. 系统会复制当前版本的所有内容
3. 新版本会自动命名为 "版本 X - 副本"

### 切换版本

1. 点击版本栏中的任意版本缩略图
2. 画布会自动加载该版本的内容
3. 当前版本会有粉色边框标识

### 自动保存

- 系统每 2 秒自动保存当前版本
- 切换版本时会自动保存并生成缩略图
- 数据保存在浏览器的 localStorage 中

## 导出功能

### 下载图片

1. 点击右上角的 **"下载"** 按钮
2. 图片会以 PNG 格式下载到本地
3. 文件名格式: `xiaohongshu-cover-{时间戳}.png`

### 复制到剪贴板

1. 点击右上角的 **"复制"** 按钮
2. 图片会复制到系统剪贴板
3. 可以直接粘贴到其他应用（如微信、小红书）

**注意**: 部分浏览器可能不支持剪贴板 API，建议使用下载功能

## 常见问题

### Q: 为什么最多只能创建 5 个版本？

A: 为了避免 localStorage 存储空间不足，限制了版本数量。如需更多版本，请先导出重要版本，然后删除不需要的版本。

### Q: 数据会丢失吗？

A: 数据保存在浏览器的 localStorage 中，只要不清除浏览器数据就不会丢失。但建议定期导出重要版本作为备份。

### Q: 可以在手机上使用吗？

A: 可以，但移动端体验可能不如桌面端。部分功能（如剪贴板复制）在移动端可能不可用。

### Q: 支持哪些图片格式？

A: 目前只支持导出 PNG 格式。PNG 格式支持透明背景，质量较高。

### Q: 可以添加图片吗？

A: 当前版本暂不支持图片上传，未来版本会添加此功能。

### Q: 字体显示不正常怎么办？

A: 默认字体已经内置到站点中。如果某个自定义 CDN 字体不可用，可以切换到本地字体或稍后重试。

## 最佳实践

### 1. 文字排版

- 标题使用较大字号（80-120）
- 正文使用中等字号（40-60）
- 注释使用较小字号（20-30）

### 2. 颜色搭配

- 背景色与文字颜色要有足够对比度
- 高亮颜色不要过多（建议 1-2 种）
- 保持整体风格统一

### 3. 版本管理

- 为每个版本命名（功能待开发）
- 定期导出重要版本
- 删除不需要的版本释放空间

### 4. 导出前检查

- 确认所有文字拼写正确
- 检查排版是否整齐
- 预览整体效果
- 确认尺寸为 1242×1660
