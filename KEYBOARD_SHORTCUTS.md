# 键盘快捷键

乔木画布支持丰富的键盘快捷键，帮助你更高效地创作海报、封面和知识卡片。

## 🛠️ 工具快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `T` | 文本工具 | 添加单行文本 |
| `R` | 矩形工具 | 添加矩形 |
| `O` | 圆形工具 | 添加圆形 |
| `L` | 线条工具 | 添加直线 |
| `Shift + L` | 箭头工具 | 添加箭头 |
| `P` | 画笔工具 | 切换自由画笔模式 |
| `V` | 选择工具 | 切换到选择模式(退出画笔) |
| `H` | 手型工具 | 切换拖拽画布模式 |
| `Space` | 临时手型工具 | 按住空格键可临时拖拽画布 |

## ✂️ 编辑快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `⌘/Ctrl + C` | 复制 | 复制选中对象到剪贴板 |
| `⌘/Ctrl + V` | 粘贴 | 从剪贴板粘贴对象 |
| `⌘/Ctrl + D` | 原地复制 | 快速复制选中对象 |
| `⌘/Ctrl + Z` | 撤销 | 撤销上一步操作 |
| `⌘/Ctrl + Shift + Z` | 重做 | 重做上一步操作 |
| `Delete` / `Backspace` | 删除 | 删除选中对象 |

## 📚 图层快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `⌘/Ctrl + ]` | 上移一层 | 将选中对象向上移动一层 |
| `⌘/Ctrl + [` | 下移一层 | 将选中对象向下移动一层 |
| `⌘/Ctrl + Shift + ]` | 置于顶层 | 将选中对象移到最顶层 |
| `⌘/Ctrl + Shift + [` | 置于底层 | 将选中对象移到最底层 |

## 🎨 其他快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `⌘/Ctrl + Shift + D` | 下载对象 | 下载选中对象为图片 |
| `Tab` | AI图片转换 | 打开AI图片转换对话框 |
| `Esc` | 退出/关闭 | 退出编辑模式或关闭对话框 |
| `?` | 快捷键帮助 | 显示快捷键帮助面板 |
| `⌘/Ctrl + /` | 快捷键帮助 | 显示快捷键帮助面板 |

## 💡 使用技巧

### 1. 快速创建形状
- 按 `R` 立即创建矩形
- 按 `O` 立即创建圆形
- 按 `L` 立即创建线条
- 按 `Shift + L` 立即创建箭头
- 无需点击工具栏,更快捷

### 2. 画笔工具
- 按 `P` 启用画笔模式
- 在画布上自由绘制
- 再按 `P` 或按 `V` 退出画笔模式
- 适合手绘标注、涂鸦等场景

### 3. 复制粘贴工作流
- `⌘/Ctrl + C` 复制对象
- 可以多次 `⌘/Ctrl + V` 粘贴
- 每次粘贴会自动偏移,避免重叠

### 4. 原地复制
- `⌘/Ctrl + D` 快速复制
- 适合需要立即创建一个副本的场景
- 比复制粘贴更快

### 5. 拖拽画布
- 按住 `Space` 临时切换到手型工具
- 松开 `Space` 恢复之前的工具
- 或按 `H` 锁定手型工具模式

### 6. 图层管理
- 使用 `⌘/Ctrl + [` 和 `]` 快速调整图层顺序
- 加上 `Shift` 可以直接置顶或置底
- 比右键菜单更高效

### 7. 文本编辑
- 双击文本进入编辑模式
- 在编辑模式下,`Delete`/`Backspace` 删除文字
- 在选择模式下,`Delete`/`Backspace` 删除整个对象
- 按 `Esc` 退出编辑模式

### 8. 多选操作
- 框选或 `Shift + 点击` 选中多个对象
- 所有快捷键都支持多选
- 复制、删除、图层调整都可以批量操作

## 🎯 设计理念

我们的快捷键设计参考了 Figma、Sketch 等专业设计软件的最佳实践:

- **单字母快捷键**: 工具切换使用单字母,无需修饰键,更快捷
- **标准快捷键**: 复制粘贴等使用系统标准快捷键,符合用户习惯
- **智能检测**: 在输入框中自动禁用快捷键,不干扰文本编辑
- **视觉反馈**: 操作后有Toast提示,让用户知道发生了什么

## 📝 注意事项

1. **输入框保护**: 在文本编辑模式或输入框中,工具快捷键会自动禁用
2. **大小写不敏感**: `T` 和 `t` 效果相同,无需切换大小写
3. **平台兼容**: 自动识别 Mac 和 Windows,使用对应的修饰键
4. **快捷键冲突**: 如果与浏览器快捷键冲突,我们的快捷键会被阻止

## 🆘 获取帮助

在应用中按 `?` 或 `⌘/Ctrl + /` 可以随时查看快捷键帮助面板。
