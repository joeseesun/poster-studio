## ✂️ 编辑快捷键测试

### Cmd/Ctrl + C - 复制
- [ ] 选中一个对象
- [ ] 按 `Cmd+C` (Mac) 或 `Ctrl+C` (Windows)
- [ ] 应该显示"已复制"Toast提示
- [ ] 控制台应该输出复制成功的日志

### Cmd/Ctrl + V - 粘贴
- [ ] 先复制一个对象 (`Cmd+C`)
- [ ] 按 `Cmd+V` (Mac) 或 `Ctrl+V` (Windows)
- [ ] 应该粘贴一个新对象
- [ ] 新对象应该偏移20px(右下方)
- [ ] 新对象应该自动被选中
- [ ] 可以多次按 `Cmd+V` 继续粘贴

### Cmd/Ctrl + D - 原地复制
- [ ] 选中一个对象
- [ ] 按 `Cmd+D` (Mac) 或 `Ctrl+D` (Windows)
- [ ] 应该立即创建一个副本
- [ ] 副本应该偏移20px(右下方)
- [ ] 副本应该自动被选中

### Cmd/Ctrl + Z - 撤销
- [ ] 执行一些操作(添加对象、移动等)
- [ ] 按 `Cmd+Z` (Mac) 或 `Ctrl+Z` (Windows)
- [ ] 应该撤销上一步操作

### Cmd/Ctrl + Shift + Z - 重做
- [ ] 先撤销一个操作
- [ ] 按 `Cmd+Shift+Z` (Mac) 或 `Ctrl+Shift+Z` (Windows)
- [ ] 应该重做上一步操作

### Delete/Backspace - 删除
- [ ] 选中一个对象
- [ ] 按 `Delete` 或 `Backspace` 键
- [ ] 对象应该被删除
