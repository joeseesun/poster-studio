# 更新日志 / Changelog

## [v1.2.0] - 2025-10-21

### ✨ 新增功能

#### 🎨 模板系统全面升级
- **引用卡片模板库**: 新增 26 个精美的引用卡片模板，覆盖所有常用画布尺寸
  - 3:4 竖版（4个尺寸）：720×960, 768×1024, 960×1280, 1080×1440
  - 1:1 方形（3个尺寸）：720×720, 960×960, 1080×1080
  - 4:3 横版（4个尺寸）：960×720, 1024×768, 1280×960, 1440×1080
  - 16:9 横版（3个尺寸）：1280×720, 1920×1080, 2560×1440
  - 9:16 竖版（3个尺寸）：720×1280, 1080×1920, 1440×2560
  - 21:9 超宽（5个尺寸）：1680×720, 2100×900, 2560×1080, 3440×1440, 5120×2160

- **模板设计特点**:
  - 使用抖音美好体，视觉效果更佳
  - 引号装饰：400 字号，醒目大气
  - 装饰短线：17 粗细，精致美观
  - 所有元素可选中编辑（背景、引号、文字、装饰线）

#### 💾 另存为模板功能优化
- **独立按钮**: 从菜单中移出，放在顶部工具栏"模板"按钮右侧
- **简化流程**: 只需填写模板名称，自动归类为"自定义"
- **快捷操作**: 支持回车键快速保存

#### 📚 模板库交互优化
- **默认显示全部**: 打开模板库时默认显示所有尺寸的模板
- **按需过滤**: 点击"只看匹配比例"按钮才过滤只显示与当前画布比例匹配的模板
- **分类优化**: "自定义"分类提前到第二位，方便快速访问用户保存的模板
- **按尺寸分类**: 模板按画布尺寸分类，而非内容类型

#### 🖌️ 画笔工具独立化
- **独立按钮**: 从"形状"对话框中移出，成为左侧工具栏的独立工具
- **位置**: 放在"多行文本"和"图片"之间
- **快捷键**: 保持 `P` 键启用，`V` 或 `Esc` 键退出

#### 📸 Unsplash 图片搜索集成
- **免费高质量图片**: 集成 Unsplash API，提供海量免费商用图片
- **智能搜索**: 支持中英文关键词搜索
- **常用关键词**: 预设常用搜索关键词，一键快速搜索
- **最近搜索**: 自动记录最近 3 个搜索关键词
- **随机图片**: 每次加载 10 张随机精选图片，支持"换一批"
- **分页浏览**: 搜索结果支持分页浏览

#### 🔧 调试功能隐藏
- **快捷键启用**: 按 `Ctrl + Option + Command + 4` 显示/隐藏调试功能
- **隐藏功能**: 清除缓存、重新生成缩略图等调试按钮默认隐藏
- **避免误操作**: 普通用户不会看到调试功能，避免误操作

## 使用说明

### 如何使用模板
1. 点击顶部工具栏的"模板"按钮
2. 浏览或搜索模板
3. 点击模板即可应用到当前画布
4. 所有元素都可以选中并编辑

### 如何保存自定义模板
1. 设计好你的画布
2. 点击"另存为模板"按钮
3. 输入模板名称
4. 按回车或点击保存

### 如何使用画笔工具
1. 点击左侧工具栏的画笔图标
2. 或按 `P` 键启用画笔模式
3. 在画布上自由绘制
4. 按 `V` 或 `Esc` 键退出画笔模式

### 如何搜索图片
1. 点击左侧工具栏的"图片"按钮
2. 切换到"网络搜图"标签
3. 输入关键词搜索或点击常用关键词
4. 点击图片即可插入到画布
