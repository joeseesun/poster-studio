## 📋 测试场景

### 场景 1: 矩形图形属性测试

**步骤:**
1. 按 `R` 键或点击形状工具,添加一个矩形
2. 确认右侧显示"图形属性"面板

**测试项:**

**填充颜色:**
- [ ] 点击填充颜色块,打开颜色选择器
- [ ] 选择预设颜色,矩形填充颜色立即改变
- [ ] 输入自定义颜色值(如 #FF5722),矩形填充颜色改变
- [ ] 关闭颜色选择器,颜色保持

**描边颜色:**
- [ ] 点击描边颜色块,打开颜色选择器
- [ ] 选择预设颜色,矩形描边颜色立即改变
- [ ] 输入自定义颜色值,矩形描边颜色改变

**描边粗细:**
- [ ] 拖拽滑块,矩形描边粗细实时改变
- [ ] 在输入框输入 `10`,按回车,描边变为10px
- [ ] 输入 `-5`,失焦后自动恢复到之前的值
- [ ] 输入 `100`,失焦后自动恢复到之前的值(最大50)

**透明度:**
- [ ] 拖拽滑块,矩形透明度实时改变
- [ ] 在输入框输入 `50`,按回车,透明度变为50%
- [ ] 输入 `0`,矩形完全透明
- [ ] 输入 `100`,矩形完全不透明

**圆角:**
- [ ] 拖拽圆角滑块,矩形四个角实时变圆
- [ ] 在输入框输入 `20`,按回车,圆角变为20px
- [ ] 输入 `0`,矩形变为直角
- [ ] 输入 `50`,矩形变为圆角矩形

**预期结果:**
- ✅ 所有属性修改实时生效
- ✅ 输入验证正确
- ✅ 超出范围自动恢复
