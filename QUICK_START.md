## 🎨 使用流程

### 添加文字
1. 点击左侧工具栏"文本"图标（T）
2. 画布上出现可编辑文字
3. 双击编辑内容

### 切换字体
1. 选中文字
2. 在右侧字体网格点击目标字体
3. 立即应用

### 应用高亮
1. 选中文字
2. 在右侧点击高亮样式（荧光笔/下划线/边框）
3. 点击颜色选择

### 调整字号
1. 选中文字
2. 在右侧拖动字号滑块（20-120）

### 更改颜色
1. 选中文字
2. 在右侧点击颜色选择器
3. 选择颜色

### 缩放画布
1. 使用底部缩放控制
2. 点击 + / - 按钮
3. 或点击百分比重置

### 版本管理
1. 点击顶部"版本 1 ▼"
2. 选择其他版本切换
3. 或点击"新建版本"/"复制当前版本"

### 导出
1. 点击顶部"导出"按钮
2. 自动下载 PNG 文件
3. 或点击"分享"复制到剪贴板

## 🔧 技术细节

### 字体加载策略
- 默认加载：思源黑体、思源宋体、Arial
- 懒加载：其他 9 个字体
- 首次点击时加载，显示加载动画
- 加载后缓存，下次立即应用

### 自动保存
- 每 2 秒自动保存到 localStorage
- 刷新页面不丢失数据
- 版本切换时保存当前版本

### 画布缩放算法
```typescript
const scaleByWidth = containerWidth / CANVAS_WIDTH;
const scaleByHeight = containerHeight / CANVAS_HEIGHT;
const autoScale = Math.min(scaleByWidth, scaleByHeight, 1);
const finalScale = (autoScale * manualScale) / 100;
```

### 最佳实践
1. **先选字体**：在添加文字前先选好字体
2. **善用版本**：重要修改前复制版本
3. **定期导出**：避免浏览器清理 localStorage
4. **合理缩放**：100% 查看细节，50% 查看整体

- 点击空白：取消选择
